# Homin Backend

O sistema Homin+ foi desenvolvido com arquitetura modular, separando as camadas de front-end e back-end em repositórios distintos.

O back-end, construído em Python com FastAPI, adota o padrão RESTful e utiliza uma estrutura organizada em módulos (API, Models, Schemas e Services), garantindo manutenção simplificada e integração eficiente com o banco de dados MySQL.
